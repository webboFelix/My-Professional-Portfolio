# UI Design

## Theme

- Background: `#050a0f`
- Primary accent: neon green `#00ff9d`
- Secondary: cyan `#00d4ff`
- Fonts: JetBrains Mono (body), Orbitron (headings)

## Patterns

- **Glass cards** — frosted panels with neon borders
- **Grid + hex background** — depth without clutter
- **Terminal boot** — first-visit narrative on homepage
- **SOC dashboard** — stats, radar chart, threat map, activity feed
- **Live logs** — monospace stream with level colors

## Customization

1. Edit `frontend/lib/api.ts` — skills, certs, timeline
2. Replace GitHub/LinkedIn in `app/contact/page.tsx`
3. Add images/video under `frontend/public/assets/`
